# Employee-Attendance-Payroll-System
The Employee Attendance and Payroll System is a Python-based application that uses a MySQL database to manage employee records, track attendance, and calculate salaries efficiently. The system allows users to add employee details such as name, department, and salary per day, and mark daily attendance as present or absent. All data is stored securely in the database, ensuring proper record management.

A key feature of this system is automatic salary calculation based on attendance. The system counts the number of days an employee is present and multiplies it by their daily salary to generate total earnings. It also provides useful features like viewing employee details, attendance records, and generating department-wise reports, making it easier to manage workforce data.

Overall, this project demonstrates how Python and database integration can simplify employee management tasks. It reduces manual effort, improves accuracy in payroll processing, and can be further enhanced with features like a graphical interface, leave management, and advanced reporting tools.
